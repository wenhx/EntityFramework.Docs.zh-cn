## <a name="potential-memory-leak-in-client-evaluation"></a><span data-ttu-id="e568a-131">客户端评估中潜在的内存泄漏</span><span class="sxs-lookup"><span data-stu-id="e568a-131">Potential memory leak in client evaluation</span></span>
